PROJECT TITLE: Gadget Shop

PURPOSE OF PROJECT: 
This project is a JavaFX gadget shop system made in BlueJ. It lets the user add gadgets to a list, display all gadgets, make a call using a mobile phone object, and download music using a MP3 object.

VERSION and DATE: Version 1.2 and 19th April 2026

HOW TO START THIS PROJECT:
1. Open the BlueJ project.
2. Compile all classes.
3. Right-click on the GadgetShop class.
4. Select the main method to run the program.

AUTHORS: Ans2303

USER INSTRUCTIONS:
1. Enter the gadget details into the text fields.
2. Click "Add Mobile" to add a new mobile phone to the list.
3. Click "Add MP3" to add a new MP3 player to the list.
4. Click "Display All" to show all gadgets in the list.
5. To make a call, enter the display number, phone number, and duration, then click "Make Call".
6. To download music, enter the display number and download size, then click "Download Music".
7. Click "Clear" to clear all text fields and the output area.
8. Click "Exit" to exit the application.
